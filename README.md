# Guille Theme

A vibrant dark theme for VS Code with green and yellow accents.

## Colors

| Token   | Hex      |
| ------- | -------- |
| Keyword | `#67FF30` |
| String  | `#C1FF30` |
| Punct   | `#64748B` |
| Symbols | `#FDE047` |
| Base    | `#EFEFF1` |
| Soft    | `#ADB6C4` |

## Install

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for "Guille Theme"
4. Click Install

## Publishing

```bash
npm install -g @vscode/vsce
vsce package
vsce publish
```
